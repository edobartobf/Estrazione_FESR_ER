<objective>
Add environment-variable overrides to scraper.py so GitHub Actions can parametrize the scraper without modifying the file.

Purpose: The Actions workflow needs to inject KEYWORD, DATA_DA, DATA_A (and optionally ANNI, CARTELLA_OUTPUT)
at run time. Rather than a CLI argument parser (which would require restructuring), a small applysecrets()
function reads FESR_* env vars and updates the module-level globals before main() runs.

Output: scraper.py with applysecrets() and `import os` at the top; main() calls applysecrets() as its
first statement.
</objective>

    applysecrets():
    - Uses `global` to declare KEYWORD, DATA_DA, DATA_A, ANNI, CARTELLA_OUTPUT
    - Reads os.environ.get("FESR_KEYWORD") — if truthy, sets KEYWORD
    - Reads os.environ.get("FESR_DATA_DA") — if the value is a non-empty string, sets DATA_DA; if the key is present but the value is empty string, sets DATA_DA = None (allows unsetting via env)
    - Same logic for FESR_DATA_A → DATA_A
    - Reads os.environ.get("FESR_ANNI") — if truthy, sets ANNI
    - Reads os.environ.get("FESR_OUTPUT") — if truthy, sets CARTELLA_OUTPUT
    - No return value; no side effects beyond updating globals

    main():
    - applysecrets() is the very first statement, before folder = Path(CARTELLA_OUTPUT)
  </behavior>
  <action>
    1. Add `import os` to the imports at the top of scraper.py (after `from datetime import datetime`).

    2. Define applysecrets() as a module-level function. Place it between the DATA_DA/DATA_A config
       vars and leggianni(). The function body:
       - global declaration for KEYWORD, DATA_DA, DATA_A, ANNI, CARTELLA_OUTPUT
       - For FESR_KEYWORD: val = os.environ.get("FESR_KEYWORD"); if val: KEYWORD = val
       - For FESR_DATA_DA: val = os.environ.get("FESR_DATA_DA"); if val is not None: DATA_DA = val or None
         (empty string env var → DATA_DA = None; non-empty → DATA_DA = val)
       - Same pattern for FESR_DATA_A → DATA_A
       - For FESR_ANNI: val = os.environ.get("FESR_ANNI"); if val: ANNI = val
       - For FESR_OUTPUT: val = os.environ.get("FESR_OUTPUT"); if val: CARTELLA_OUTPUT = val

    3. In main(), add `applysecrets()` as the first line (before `folder = Path(CARTELLA_OUTPUT)`).
  </action>
