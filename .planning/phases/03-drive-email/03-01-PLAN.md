<objective>
Create upload_drive.py that uploads the scraper output folder to Google Drive using a Service Account.

Purpose: AUTO-03 requires all CSV/JSON+PDF outputs to be uploaded to a Drive folder after each run.
The Service Account JSON comes from the GDRIVE_SA_JSON secret (a JSON string in the env var).
The script must fail explicitly if required secrets are absent (AUTO-05).

Output: upload_drive.py that can be run as `python upload_drive.py` in CI.
</objective>

<tasks>

<task type="auto" tdd="true">
  <name>Task 1: Create upload_drive.py</name>
  <files>upload_drive.py</files>
  <behavior>
    Imports: json, os, sys, pathlib.Path
    From google: google.oauth2.service_account.Credentials, googleapiclient.discovery.build,
                 googleapiclient.http.MediaFileUpload

    SCOPES = ["https://www.googleapis.com/auth/drive.file"]

    upload_folder(folder_path, folder_id, sa_json_str):
      - Parses sa_json_str with json.loads()
      - Creates Credentials from service_account_info with SCOPES
      - Builds Drive v3 service
      - Iterates folder_path.rglob("*"), skipping directories
      - For each file: creates file in Drive with name=f.name, parents=[folder_id]
        using MediaFileUpload with resumable=True
      - Prints "Caricato: {f.name}" for each file
      - Returns list of uploaded file metadata dicts

    main():
      - sa_json = os.environ.get("GDRIVE_SA_JSON")
      - folder_id = os.environ.get("GDRIVE_FOLDER_ID")
      - output_folder = os.environ.get("FESR_OUTPUT", "data")
      - If not sa_json: print error to stderr, sys.exit(1)
      - If not folder_id: print error to stderr, sys.exit(1)
      - Call upload_folder(Path(output_folder), folder_id, sa_json)
      - folder_url = f"https://drive.google.com/drive/folders/{folder_id}"
      - Print summary: f"Caricati {len(uploaded)} file su Drive: {folder_url}"
      - Path("drive_url.txt").write_text(folder_url, encoding="utf-8")

    if __name__ == "__main__": main()
  </behavior>
  <action>

  </action>
