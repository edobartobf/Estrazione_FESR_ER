and Drive folder link. Must work for zero-result runs too. All credentials via env vars (AUTO-05).

</objective>

  <name>Task 1: Create notify_email.py</name>
  <files>notify_email.py</files>
  <behavior>
    Imports (stdlib only): csv, os, smtplib, sys, pathlib.Path,
    email.mime.multipart.MIMEMultipart, email.mime.text.MIMEText

    build_summary(output_folder):
      - Reads all files matching output_folder / "delibere_*.csv"
      - Aggregates rows from all matching CSVs
      - Returns dict: {"count": int, "azioni": sorted list of unique azione strings (non-empty)}
      - If no CSV files found: returns {"count": 0, "azioni": []}
      - Azioni extracted from the "azioni" field (semicolon-separated values per row)

    send_email(smtp_host, smtp_port, smtp_user, smtp_pass, recipients, subject, body):
      - Creates SMTP connection to smtp_host:smtp_port
      - Calls starttls()
      - Logs in with smtp_user / smtp_pass
      - Builds MIMEMultipart message with MIMEText body (plain, utf-8)
      - Sends to all recipients
      - Does NOT suppress exceptions — let them propagate

    main():
      - Read required secrets from env: SMTP_HOST, SMTP_USER, SMTP_PASS, EMAIL_RECIPIENTS
        For each missing one: print "ERROR: {name} non impostato" to stderr, sys.exit(1)
      - smtp_port = int(os.environ.get("SMTP_PORT", "587"))
      - output_folder = os.environ.get("FESR_OUTPUT", "data")
      - drive_url = Path("drive_url.txt").read_text(encoding="utf-8").strip()
        if Path("drive_url.txt").exists() else "N/D"
      - summary = build_summary(output_folder)
      - recipients = [r.strip() for r in EMAIL_RECIPIENTS.split(",") if r.strip()]
      - subject = f"FESR Scraper — {summary['count']} delibere trovate"
      - body = multiline plain text with:
          - "Delibere trovate: {count}"
          - "Azioni identificate: {azioni joined by ', ' or 'nessuna'}"
          - "Cartella Drive: {drive_url}"
      - Call send_email(...)
      - Print confirmation: f"Email inviata a {len(recipients)} destinatari."

    if __name__ == "__main__": main()
  </behavior>
  <action>

    SMTP_PORT is optional (default 587).
  </action>
